__attribute__((constructor)) 构造函数属性,可以在main()函数之前被自动调用,同样的,析构函数属性使函数在main()完成或者调用exit()后自动被调用
具有这些属性的函数对于初始化在程序执行期间被隐式的使用的数据很有用

web 的cgi handler函数通过CYG_HTTPD_HANDLER_TABLE_ENTRY宏,生成一个web_root的链表结构 
#define CYG_HTTPD_HANDLER_TABLE_ENTRY(_name_, _path_, _handler_) \
    web_handler_t _name_ = {_path_, _handler_, false, false};    \
    void __attribute__((constructor)) _name_##_constructor() {   \
        _name_.next = web_root;                                  \
        web_root = &_name_;                                      \
    }
