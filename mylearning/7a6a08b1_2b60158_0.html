递归将add或者删除等文件操作撤销,撤销当前修改,(恢复到服务器上的状态)
svn revert -R .

提交修改
svn commit -m ""
svn commit -m "abc" a.c b.c filename
svn commit 用于提交代码的修改-m 是message意思，用于在""中写本次上传原因。

如果有需要添加文件的需要首先执行如下命令：
svn add filename
其中add为增加的意思,filename为要增加的文件夹名。

上传库以及隐藏文件
svn add --no-ignore --force
svn add * --no-ignore --force
该命令强制上传所有文件。

删除命令
svn rm filename
rm为删除命令filename为要删除的文件。

更新到指定版本
svn up -r 版本号

svn revert --recursive filename

svn log 获取解决问题和特性开发列表 在导入exel/csv
1. 导出指定时间日志的xml文件格式，便于我们解析
svn log -r {2020-1-1}:{2021-12-31} --xml > svnlog.xml
2. 写一个python脚本，解析xml文件为csv格式 
#coding:utf-8
import os
import hashlib
import struct
import sys
import gzip, io

import xml.etree.ElementTree as ET

tree = ET.parse("svnlog.xml")

#根节点
root = tree.getroot()

#标签名
print('root_tag:', root.tag)
for log in root:
    #属性值
    print("name:", log.attrib["revision"])
    #标签中内容
    print("author:",log[0].text)
    print("date:",log[1].text)
    print("msg:",log[2].text)
